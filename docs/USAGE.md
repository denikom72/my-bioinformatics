USAGE
-----
- Install Biopython for Python scripts: python3 -m pip install biopython
- Perl scripts require BioPerl installed (legacy)
- Java modules are plain Java classes; compile with javac in the biojava_module directory

Example runs:
python3 biopython_rewrite/gc_content.py legacy_data/example.fasta
perl legacy_perl/gc_content.pl legacy_data/example.fasta
